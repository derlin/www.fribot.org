== Description ==

**A JQuery based NewFlash Displays a scroll up text list of news or information and ads or link. Starting from Wordpress ver.3.0 the ticker can also be used as a image gallery with description**

in NewsTicker Setting, you can set:

* NewsTicker Title: Title for NewsTicker.
* NewsTicker Height: Height for NewsTicker.
* NewsTicker Speed: Speed for NewsTicker in second.
* NewsTicker Title Text Align: Text align for title of NewsTicker.
* NewsTicker Content Text Align: Text align for content of NewsTicker.

in NewsTicker Create New pages, you can set:

* NewsTicker Title: Title for your NewsTicker.
* NewsTicker Content: Content for your NewsTicker, HTML is allowed.
* NewsTicker Status: Status for your NewsTicker. If you do not want the NewsTicker live, select "Inactive".
* NewsTicker Show for: Show the NewsTicker for selected user.


== Installation ==

1. Upload the folder `NewsTicker-Aink` to your `/wp-content/plugins/` directory,
2. Activate the plugin through the "Plugins" menu in WordPress,
3. Configure options via the NewsTicker ==> Settings menu,
4. Create a new NewsTicker on NewsTicker ==> Create New page.
5. Go to Appearance ==> Widget and drag 'n drop your NewsTicker Aink to your sidebar to show NewTicker if you have sidebar.
